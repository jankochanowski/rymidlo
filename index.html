Teoria

Wersja robocza silnika wyszukiwarki rymów, który w przeciwieństwie do internetowych słowników rymów (w jęz. polskim) potrafi samodzielnie obliczyć miejsce padania akcentu i dzięki temu wyznaczyć przestrzeń rymową (część słowa, która powinna się rymować). Oblicza też liczbę sylab i wykrywa (z marginesem błędu), w jakich przypadkach samogłoska i nie pełni funkcji zgłoskotwórczej (jest to niezbędne, aby prawidłowo wyznaczyć obszar współdźwięczności). Docelowo silnik będzie również zwracał tablicę wszystkich wariantów homofonicznych danego rymu, uwzględniając m.in. zjawisko ubezdźwięcznienia wygłosu czy pary głosek różnych ortograficznie, ale identycznie brzmiących (ó i u, rz i ż, ch i h). Silnik jest dodatkowo rozbudowywany o analizator wersyfikacyjny.

Krok 1: Gdzie szukać rymu?

Rym to „powtórzenie jednakowych lub podobnych układów brzmieniowych w zakończeniach wyrazów zajmujących określoną pozycję w obrębie wersu (w wierszu) lub zdania (w prozie)” [STLSłownik terminów literackich, red. M. Głowiński, T. Kostkiewiczowa, A. Okopień-Sławińska, J. Sławiński, Wrocław 1998.].

Załóżmy dla uproszczenia, że ta „określona pozycja w obrębie wersu” to pozycja końcowa, ponieważ w większości wierszy występują rymy końcowe. Pierwszy krok jest więc prosty - należy znaleźć ostatni wyraz w danym ciągu znaków (wersie):

function findLastWord(str) {

    return splitedVerse[last];
}


Krok 2: Jak określić liczbę sylab w słowie?

OK, mamy ostatnie słowo. Która część tego słowa stanowi przestrzeń rymową (obszar współdźwięczności, czyli po prostu obszar, który ma się rymować)? Oczywiste jest, że nie całość słowa, ponieważ wtedy mówilibyśmy o rymach tautologicznych (para zdrowie - zdrowie nie jest rymem, bo jest fonetycznie dokładnie taka sama).

Przestrzeń rymową stanowi jakaś końcowa część słowa. Ta część jest w różnych słowach zmienna - zależy od akcentu, ponieważ obejmuje samogłoskę akcentowaną i wszystkie głoski po niej następujące. Przestrzeń rymową mogą stanowić zatem np. następujące ciągi znaków (zaznaczone podkreśleniem): dno, dom, Radom, tragedia, fizyka, itd. Aby określić przestrzeń rymową, musimy znać liczbę sylab w słowie. Jak ją policzyć?

Po pierwsze, musimy wiedzieć, że ośrodkiem sylaby mogą być wyłącznie samogłoski (wyłącznie samogłoski mogą być zgłoskotwórcze). Operacja przeliczenia samogłosek (a, e, i, o, u, y, ą, ę) jednak nie wystarczy, ponieważ nie wszystkie samogłoski w słowie pełnią funkcję zgłoskotwórczą. Problematyczna jest głoska „i”, która może pełnić 1) tylko funkcję zmiękczającą lub 2) funkcje zmiękczającą i zgłoskotwórczą, np.: kwitek (2), kwiatek (1), wariat (1), diariusz (1). Zauważmy, że wszystkie wymienione słowa są dwusylabowe, a jednak ich liczba samogłosek jest różna - to odpowiednio: 2, 3, 3, 4.

Trzeba więc znaleźć warunki, w których „i” nie pełni funkcji zgłoskotwórczej. Samogłoska „i” nie jest zgłoskotwórcze wtedy, kiedy inna samogłoska pełni tę funkcję, a więc wtedy, kiedy występuje w jednej z następujących konfiguracji: ia, ie, ii, io, iu, ią, ię.

Żeby obliczyć liczbę sylab w słowie, musimy zatem zliczyć wszystkie samogłoski i odjąć liczbę wystąpień każdej konfiguracji: ia, ie, ii, io, iu, ią, ię:

//vowel or consonant check - auxiliary function
var isVowelCheck = function(letter) {
        var vowels = ["a", "e", "i", "o", "u", "y", "ą", "ę"];
        return vowels.includes(letter);
    }
    //count syllables in a verse;
function countSyllables(str) {
    var splittedPhrase = str.split("");
    var numberOfSyllables = 0;
    for (var i = splittedPhrase.length - 1; i >= 0; i--) {
        if (isVowelCheck(splittedPhrase[i]) == true) {
            numberOfSyllables += 1;
        }
    }
    if (str.length > str.replace(/ia/g, "").length) {
        var x = str.length - str.replace(/ia/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/ie/g, "").length) {
        var x = str.length - str.replace(/ie/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/ii/g, "").length) {
        var x = str.length - str.replace(/ii/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/io/g, "").length) {
        var x = str.length - str.replace(/io/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/iu/g, "").length) {
        var x = str.length - str.replace(/iu/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/iy/g, "").length) {
        var x = str.length - str.replace(/iy/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/ią/g, "").length) {
        var x = str.length - str.replace(/ią/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    if (str.length > str.replace(/ię/g, "").length) {
        var x = str.length - str.replace(/ię/g, "").length;
        numberOfSyllables = numberOfSyllables - x / 2;
    }
    return numberOfSyllables;
}


Krok 3: Jak określić typ rymu?

Powiedzieliśmy już, że rym związany jest z akcentem. Dla języka polskiego charakterystyczny jest akcent paroksytoniczny (na 2. sylabę od końca); akcent oksytoniczny (na ostatnią sylabę) dotyczy natomiast przede wszystkim wyrazów jednosylabowych.

Ze względu na rodzaj występującego w rymie akcentu, wyróżniamy rymy męskie (albo: oksytoniczne) i rymy żeńskie (albo: paroksytoniczne). Warto wiedzieć, że istnieją również rymy głębsze - daktyliczne (albo: proparoksytoniczne), czyli obejmujące obszar od 3. sylaby od końca. Zazwyczaj (choć nie tylko) wiążą się one ze słowami zapożyczonymi, akcentowanymi na 3. sylabę od końca (np. fizyka, matematyka). Wykrycie rymów proparoksytonicznych wymagałoby dołączenia słownika zawierającego listę leksykalnych rdzeni wyrazów akcentowanych proparoksytonicznie (a także m.in. zakończeń czasowników w 1. i 2. os. l. mn. czasu przeszłego (-liśmy, -liście, -łyśmy, -łyście), ponieważ formy: zrobiliśmy, zrobiliście itd. również akcentujemy proparoksytonicznie), więc ten problem na razie pomińmy.

Wróćmy do rymów męskich i żeńskich. W sposób męski mogą się rymować słowa, które zawierają albo jedną samogłoskę, albo dwie samogłoski, jeśli jedna z samogłosek nie jest zgłoskotwórcza. W sposób żeński mogą się rymować słowa, które nie rymują się w sposób męski (czyli te, które nie spełniają ww. warunku).
